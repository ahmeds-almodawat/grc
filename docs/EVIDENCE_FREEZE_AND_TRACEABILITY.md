# Evidence Freeze and Traceability

## Freeze rule

After final approval files are completed, avoid changing unrelated release artifacts before running final proof.

## Traceability rule

Every final go/no-go decision should be traceable to:

- Approval JSON files
- `v674:signoff-check`
- `v674:sync-manual-evidence`
- `v66:strict-proof`
- `proof:all`

## Change control

If any technical file changes after signoff, rerun proof and document why the change was necessary.
