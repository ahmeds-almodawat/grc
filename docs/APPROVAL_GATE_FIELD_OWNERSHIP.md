# Approval Gate Field Ownership

## Pilot signoff ownership

| Section | Owner | Purpose |
| --- | --- | --- |
| `management_admin` | Management/Admin | Approves controlled internal pilot scope |
| `it` | IT | Confirms technical readiness and evidence review |
| `quality` | Quality | Confirms controlled-pilot quality and OVR boundary |
| `pilot_scope` | Management/Admin | Defines exact controlled pilot scope |
| `maximum_pilot_users` | Management/Admin / IT | Confirms pilot user cap is between 1 and 15 |
| `reviewed_evidence` | IT / Quality | Confirms review of proof artifacts |

## Confidentiality ownership

| Section | Owner | Purpose |
| --- | --- | --- |
| `it_reviewer` | IT | Confirms technical confidentiality boundary |
| `quality_reviewer` | Quality | Confirms OVR/patient confidentiality boundary |
| `statements` | IT / Quality | Confirms no real patient data or confidential OVR details |
| `scope` | IT / Quality | Confirms controlled internal pilot scope |
