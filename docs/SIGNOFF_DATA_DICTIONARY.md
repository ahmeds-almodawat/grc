# Signoff Data Dictionary

Defines required approval field groups: Management/Admin, IT, Quality, pilot scope, maximum pilot users, reviewed evidence, IT confidentiality, Quality confidentiality, and confidentiality statements.

## Safety boundary

This document does not approve the pilot by itself. Real human approval must still be entered in `release/v674/approvals/pilot-signoff.json` and `release/v674/approvals/ovr-confidentiality-confirmation.json`.
