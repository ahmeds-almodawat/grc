# v5.6 Company Rollout Guide

Do not onboard 1,000 employees on one day.

Recommended waves:

1. Pilot wave: 60–80 users
2. Department manager wave: around 200 users
3. Core departments wave: around 400 users
4. Remaining company users

Each wave requires:

- user import review
- training completion
- department manager confirmation
- support owner assignment
- backup/export before wave
- blocker review
