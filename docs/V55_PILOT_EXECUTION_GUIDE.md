# v5.5 Pilot Execution Guide

Run a controlled pilot before company-wide rollout.

Recommended pilot size: 60–80 users.

Pilot groups:

- 3–5 executives
- 5–10 Governance / Quality / Audit users
- 10–15 department managers
- 30–50 employees

Pilot exit criteria:

- No critical RLS/security issue
- OVR workflow completed end-to-end
- Evidence upload and approval completed
- Export/backup package generated
- Support process confirmed
