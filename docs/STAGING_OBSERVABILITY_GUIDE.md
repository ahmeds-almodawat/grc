# Staging Observability Guide

## Minimum monitoring

- Frontend runtime errors.
- Authentication failures.
- Edge bridge failures.
- RPC failures.
- Permission-denied events.
- OVR workflow errors.
- Build/deploy status.

## Evidence

Capture screenshots or logs for:

- Successful deployment.
- Successful smoke test.
- Failed unauthorized access attempt.
- Error-free dashboard load.
- Rollback result.
