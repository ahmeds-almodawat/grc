# Pilot Day-by-Day Checklist

## Day 0 — Setup
- Confirm backup/export works.
- Confirm 10 pilot users can login.
- Confirm RLS persona test passed.
- Confirm Arabic/English toggle works.

## Day 1 — Basic use
- Create one action plan.
- Assign one milestone and one task.
- Upload one evidence file.
- Request approval.

## Day 2 — OVR workflow
- Submit one test OVR.
- Route to Supervisor/HOD.
- Submit investigation.
- Quality reviews and closes with evidence.

## Day 3 — Reporting
- Export executive report.
- Export OVR report.
- Review dashboard and heatmap.

## Day 4 — Issues
- Log all bugs.
- Classify blockers/major/minor.
- Fix blockers before wider rollout.

## Day 5 — Acceptance
- Department manager signoff.
- Quality manager signoff.
- IT/security signoff.
- Executive decision for next wave.
