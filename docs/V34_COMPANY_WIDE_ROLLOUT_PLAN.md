# v3.4 Company-wide Rollout Plan

## Recommended waves

### Wave 1 — Core governance users
60 users. Prove core workflow.

### Wave 2 — Department managers
200-300 users. Focus on department dashboards, tasks, approvals, and OVR routing.

### Wave 3 — Operational staff
500-700 users. Focus on My Work, evidence uploads, assigned tasks, and OVR submission.

### Wave 4 — Full organization
All remaining employees. Enable full notification and reporting cadence.

## Rollout gate before each wave
- No critical open issue.
- Backup package created.
- Restore dry-run status reviewed.
- RLS tests repeated.
- Help/support owner assigned.
