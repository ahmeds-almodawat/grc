# Approval Evidence Binder Policy

The evidence binder must include technical, operational, governance, confidentiality, and final signoff artifacts.

The binder supports controlled-pilot approval only. It must not be represented as production approval.

Minimum binder contents:

- Proof suite result
- Signoff check result
- Restore dry-run evidence
- Persona proof evidence
- Runtime bridge evidence
- Approval files
- OVR confidentiality confirmation
- Final proof run result
