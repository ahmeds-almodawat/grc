# Control Testing and Effectiveness Guide

## Objective

Create a controlled-pilot control testing structure that can be reviewed by management, IT, Quality, and Audit.

## Control categories

- Access control
- Segregation of duties
- Evidence retention
- OVR confidentiality
- Approval workflow integrity
- Restore integrity
- Runtime bridge access
- Report review controls

## Test result statuses

- Passed
- Passed with observation
- Failed non-blocking
- Failed blocking
- Not applicable for pilot

## Acceptance rule

Blocking failures require remediation before pilot expansion. Non-blocking observations may be carried into the remediation register.
