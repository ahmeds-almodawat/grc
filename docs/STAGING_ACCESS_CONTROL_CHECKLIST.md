# Staging Access Control Checklist

## Personas

- Admin / management reviewer.
- IT reviewer.
- Quality reviewer.
- Audit reviewer.
- Standard user.
- OVR involved user.
- Unauthorized/negative persona.

## Checks

- Role boundaries work.
- Privileged actions are not visible to regular users.
- Privileged RPCs use approved bridge paths.
- OVR confidentiality boundaries are respected.
- Audit/report access is restricted.
- Synthetic data is clearly marked.
