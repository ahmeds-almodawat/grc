# OVR Confidentiality Confirmation Guide

## ovr-confidentiality-confirmation.json

Required reviewers:

- IT reviewer
- Quality reviewer

Allowed decisions:

- `confirmed`
- `approved`
- `accepted`

Required statements must be explicitly true only after review:

- no real patient identifiers
- no confidential OVR details
- controlled pilot users only
- local staging proof reviewed

Do not use real patient identifiers or confidential OVR details in pilot proof evidence.
