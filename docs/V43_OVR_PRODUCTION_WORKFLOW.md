# v4.3 OVR Production Workflow

Purpose: make OVR usable in a real hospital workflow, not only as a form.

Workflow target:

1. Reporter submits OVR.
2. Supervisor/HOD investigates.
3. Quality confirms category and severity.
4. Major/sentinel items escalate.
5. Corrective action is created where needed.
6. Evidence is reviewed.
7. Quality Manager closes.

Production rules:

- OVR remains confidential.
- OVR is not filed in the medical record.
- Reporter cannot close the OVR.
- Department cannot independently close the OVR.
- Major or sentinel OVRs require Quality Manager review.
- Closure requires accepted evidence when evidence is required.
- Returned OVRs require a clear reason.

Migration 034 adds OVR production checklist templates and per-OVR checklist results.
