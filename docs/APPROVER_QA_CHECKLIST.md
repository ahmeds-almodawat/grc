# Approver QA Checklist

## Management / Admin

- Confirms pilot scope.
- Confirms maximum user count.
- Confirms this is a controlled internal pilot only.
- Confirms production rollout is not approved by this gate.

## IT

- Confirms technical evidence reviewed.
- Confirms restore dry-run reviewed.
- Confirms security definer audit reviewed.
- Confirms access boundaries are understood.

## Quality

- Confirms OVR confidentiality restrictions.
- Confirms no real patient identifiers are used in pilot evidence.
- Confirms no confidential OVR details are used in pilot evidence.
- Confirms controlled pilot users only.
