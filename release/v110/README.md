# v11.0 Enterprise GRC Program Suite

This release pack adds an additive enterprise GRC program layer on top of v10.0:

- Policy & document control
- Policy attestation
- Training / compliance learning
- Third-party / vendor risk
- KRI / KPI monitoring
- Regulatory change management
- Risk acceptance / exceptions
- Business impact and continuity planning
- Continuity exercises
- Audit follow-up reviews
- Board pack readiness views
- Program maturity assessments

It does not assert production readiness and does not bypass human approval gates.
